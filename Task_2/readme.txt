Т.к. по условию задачи математические операции должны производиться с числами любой длины,
то методы объектов класса LongInt: add(), sub(), mult() и div(), принимают два числа, записанных
в виде строки. 

Все методы возвращают строку, представляющую собой результат выполненной операции.

Сложение, вычитание, умножение и деление производится как с положительными, так и с отрицательными
целыми числами. 

В случае, если два числа не делятся без остатка, в консоль будет выведен результат
в формате "неполное частное (остаток: ...)".

При попытке деления на 0 возбуждается исключение "ValidationError".
