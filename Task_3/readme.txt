Чтобы не менять прототип встроенного класса "Array", объекты класса "Товар" были помещены в расширенный 
массив класса "CatalogArray", которому и был добавлен метод, позволяющий извлекать объекты в соответствии 
с запросом.

Метод select(request) класса "CatalogArray" принимает строку вида "name-contains-fd&price-=2-&quantity->5&description-ends-abc",
где параметры могут быть указаны в любой последовательности.
На выходе возвращает массив класса "CatalogArray", включающий только подходящие по запросу объекты.
